## 📈 Prophet Forecasting Code Snippet

```python
from prophet import Prophet

# Prepare Data
df = df[['Order Date', 'Sales']]
df.columns = ['ds', 'y']

# Model & Forecast
model = Prophet()
model.fit(df)
future = model.make_future_dataframe(periods=6, freq='M')
forecast = model.predict(future)

# Visualize
model.plot(forecast)
